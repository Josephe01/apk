Role-Based Access Control (RBAC) Information
Admin Role

Full system access including user management, role assignment, and access to all features and logs.

Manager Role

Can manage inventory items, view reports, and access user information but cannot modify user roles.

Viewer Role

Read-only access to inventory and basic user information. Cannot modify any data.

Security Features
  • Permission-based access: Each action is checked against user permissions
  • Audit logging: All access attempts and actions are logged
  • Session management: Secure user authentication and session handling
  • Role validation: Roles cannot be bypassed or escalated without proper authorization
